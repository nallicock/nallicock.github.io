COVID-19 Twitter Bot

Since many people were curious about cases and other statistics of COVID-19, I created this bot using Python, retrieving data from an API to post to my twitter account every 24 hours (6PM EST). This bot was uploaded to Heroku in order to run 24/7 instead of starting the script manually on my computer (click to navigate to the Twitter page).
